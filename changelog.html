Better Teacher Portal

Version Changelog
  • v2.9 Latest
    • Email Page Exclusion: Disabled script execution and custom styles on mass email pages (massEmail.do) to prevent visual layout breakage.
    • Version Changelog: Added this interactive timeline to display historical version updates, accessible by clicking the version number in the extension popup.
  • v2.8
    • Popup Enhancements: Widened the popup dashboard interface to improve text wrapping and general readability.
    • Settings Tooltips: Implemented customizable CSS info tooltips for all setting switches to explain what they control.
  • v2.7
    • Red Pin Badges: Converted static red pin images in Gradebook columns into beautiful, responsive "Posted" status badges.
    • Auto-Resizing Comment Popups: Added support for auto-resizing comment editor windows to perfectly fit contents and avoid double scrollbars.
    • Hover Highlights: Integrated row hover state highlights across all tables to assist with data entry.
  • v2.6
    • Logo Refinement: Swapped generic icons with our custom BTP leafy logo design in onboarding screens and extensions menu.
  • v2.5
    • Hot Injection: Shifted script loading to the service worker to automatically re-inject enhancements into existing tabs without requiring a page reload.
    • Ko-fi Support: Added a quick-link tip button to settings and welcome page.
  • v2.2
    • Popup Styling: Created a responsive layout for settings on low screen resolutions.
    • Table Borders: Enhanced the borders and grids in lists for better structural readability.
    • Readonly Comment Labels: Added clear informative notices to the comment editor for posted comments.
  • v2.0
    • Column Drag & Drop: Enables interactive column reordering on the main attendance grid. Column setups are automatically persisted in user storage.
    • Excused Attendance Safeguard: Prevents redundant clicks by automatically graying out and disabling "Absent" buttons for students already marked Excused in Class Attendance.
    • Observer Optimization: Enhanced the lifecycle of background and page mutation observers to prevent memory leaks and redundant calls.
  • v1.5
    • Grades Draft Protection: Captures typed comments in report card inputs and saves them locally, protecting your work from timeout or crash loss.
    • Sync Indicators: Injected a custom sidebar with font controls, a typing indicator, and sync status checks.
  • v1.4
    • Celebration Mode: Implemented a beautiful burst of confetti when clicking "Post" or "Post Grades..." to celebrate attendance and grading submissions.
    • Sticky Columns: Locked the student name column to the left of the table during horizontal scroll for easier data entry.
    • Name Formatters: Re-formatted raw student names to display first names in bold for quicker visual navigation.
